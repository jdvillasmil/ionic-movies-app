    - summary = null → returns 0
    - summary.normalCount = 0 → returns 0
    - summary = { normalTotal: 17, normalCount: 2 } → returns 8.5
    - summary = { normalTotal: 10, normalCount: 3 } → returns 3.3 (Math.round(3.333... * 10) / 10)

    criticAvg computed getter:
    - summary = null → returns 0
    - summary.criticCount = 0 → returns 0
    - summary = { criticTotal: 9, criticCount: 1 } → returns 9.0

    isLoggedIn getter:
    - authService.currentUser = null → returns false
    - authService.currentUser = { uid: 'abc' } → returns true

    ngOnInit():
    - Calls TmdbService.getDetail, ReviewService.getMediaSummary, ReviewService.getReviews in parallel (Promise.all)
    - After Promise.all, if user is logged in: calls getUserReview + getUserDoc
    - Sets isLoading = false after all data loaded

    Testing approach:
    - Create test module with providers mocking TmdbService, ReviewService, AuthService, UserService
    - Use jasmine.createSpyObj for each service with resolved Promise returns
    - Test computed getters by setting this.summary directly on the component instance

  <implementation>
    After tests are RED, implement media-detail.page.ts:

    Component class (standalone, no authGuard — route is public):

    Properties:
    - private route = inject(ActivatedRoute)
    - private tmdbService = inject(TmdbService)
    - private reviewService = inject(ReviewService)
    - private authService = inject(AuthService)
    - private userService = inject(UserService)
    - private toastController = inject(ToastController)
    - private alertController = inject(AlertController)
    - mediaType = this.route.snapshot.paramMap.get('mediaType') as 'movie' | 'tv'
    - id = Number(this.route.snapshot.paramMap.get('id'))
    - mediaId = `${this.mediaType}_${this.id}`
    - detail: MediaDetail | null = null
    - summary: MediaSummary | null = null
    - reviews: ReviewDoc[] = []
    - userReview: ReviewDoc | null = null
    - currentProfile: UserProfile | undefined
    - isLoading = true
    - isSubmitting = false
    - reviewForm = new FormGroup({ score: new FormControl(5, { validators: [Validators.min(1), Validators.max(10)], nonNullable: true }), text: new FormControl('', { validators: [Validators.required], nonNullable: true }) })

    Computed getters:
    - get isLoggedIn(): boolean { return this.authService.currentUser !== null; }
    - get normalAvg(): number { if (!this.summary || this.summary.normalCount === 0) return 0; return Math.round((this.summary.normalTotal / this.summary.normalCount) * 10) / 10; }
    - get criticAvg(): number { if (!this.summary || this.summary.criticCount === 0) return 0; return Math.round((this.summary.criticTotal / this.summary.criticCount) * 10) / 10; }

    ngOnInit():
    - const [detail, summary, reviews] = await Promise.all([this.tmdbService.getDetail(this.id, this.mediaType), this.reviewService.getMediaSummary(this.mediaId), this.reviewService.getReviews(this.mediaId)])
    - Set this.detail, this.summary, this.reviews
    - If currentUser exists: const [userReview, profile] = await Promise.all([this.reviewService.getUserReview(this.mediaId, uid), firstValueFrom(this.userService.getUserDoc(uid))]). Set this.userReview, this.currentProfile.
    - this.isLoading = false

    submitReview():
    - Build ReviewDoc from form values + currentProfile + currentUser
    - authorName from currentProfile.displayName, role from currentProfile.role (NOT Auth User)
    - Call reviewService.submitReview(mediaId, reviewDoc)
    - After success: reload summary + reviews, set userReview to the just-submitted doc
    - Show success toast via ToastController

    editReview():
    - oldScore = this.userReview.score
    - Build updated ReviewDoc with new form values, updated.updatedAt = Date.now()
    - Call reviewService.editReview(mediaId, oldScore, updated)
    - Reload summary + reviews, update this.userReview

    deleteReview():
    - Show confirmation AlertController before proceeding
    - On confirm: call reviewService.deleteReview(mediaId, this.userReview)
    - Set this.userReview = null, reload summary + reviews

    Template structure:
    - IonHeader with IonBackButton (defaultHref="/tabs/home")
    - Loading skeleton (when isLoading): IonSkeletonText placeholders for poster + title
    - Poster: <img [src]="detail.posterPath | tmdbImage:'w500'" *ngIf="detail?.posterPath"> using TmdbImagePipe
    - Title, releaseDate, overview
    - Genres: chip list using *ngFor on detail.genres
    - Dual ratings section: "Usuarios: {{ normalAvg | number:'1.1-1' }} ({{ summary?.normalCount ?? 0 }} reseñas)" and "Críticos: {{ criticAvg | number:'1.1-1' }} ({{ summary?.criticCount ?? 0 }} reseñas)"
    - Reviews list: *ngFor review in reviews — show authorName, role badge (color 'tertiary' for critic, 'primary' for normal), score badge, text
    - Auth-conditional review section (*ngIf="isLoggedIn"):
      - If !userReview: show review form (IonRange [formControl]="reviewForm.controls.score" [min]="1" [max]="10" [snaps]="true", IonTextarea [formControl]="reviewForm.controls.text", submit button calling submitReview())
      - If userReview: show edit form (pre-populate form in ngOnInit when userReview exists), edit button calling editReview(), delete button calling deleteReview()
    - If !isLoggedIn: show "Inicia sesión para escribir una reseña" prompt
    - Cast section: top 10 cast — actor photo (TmdbImagePipe 'w185'), name, character

    Standalone imports from @ionic/angular/standalone:
    IonHeader, IonToolbar, IonTitle, IonContent, IonBackButton, IonButtons, IonButton,

    IonList, IonListHeader, IonCard, IonCardContent, IonSpinner, ToastController, AlertController

    From @angular/forms: ReactiveFormsModule, FormGroup, FormControl, Validators
